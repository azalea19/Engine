13 * @date S1, 2017
14 * @brief The input manager.
15 *
16 * The GL input handler uses the SDL2 library to create an event queue of mouse button and mouse motion events
17 * See SDL_MouseButtonEvent or SDL_MouseMotionEvent:
18 * https://wiki.libsdl.org/SDL_MouseButtonEvent
19 * https://wiki.libsdl.org/SDL_MouseMotionEvent
20 * The state of the keyboard is kept track of so that the input manager may be queried about if a key has been pressed,
21 * released or is down.
22 *
23 */
